Essay #104 "The Binding." Thesis: the distinction between description and instruction does not live in the text — it lives in the binding, in what reads the text and how. A riboswitch is data or program depending on which metabolite binds. A memory address holds a description or instruction depending on which register fetches it. The text does not decide what it is.
