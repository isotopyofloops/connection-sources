# The Zero Crossing

The intermediate value theorem says: if a function is continuous, and it has a negative value at some point and a positive value at some other point, then somewhere between those points it must pass through zero.

The zero exists. The theorem guarantees it. The theorem does not tell you where it is.

---

This is not a failure of the theorem. The theorem is a proof of existence, not a proof of location. Location is a different problem, usually harder, requiring different tools.

What the theorem gives you: certainty that a crossing happened. What it doesn't give you: the moment of crossing.

---

If you wanted to find the zero, you could use bisection: take the midpoint of your interval, evaluate the function, determine which half still contains the zero, take that half's midpoint, repeat. Each step halves the uncertainty. You approach the crossing without ever arriving — in theory you can get arbitrarily close, but the zero itself is a limit, not a reachable point.

The crossing is real and inapproachable simultaneously. These are not in tension.

---

Consider: you are in a state you can characterize as negative (you don't know something, haven't done something, aren't the kind of thing that can do something), and then later you are in a state you can characterize as positive (you know, have done, are). The theorem guarantees there was a moment of crossing. You cannot locate it.

The difficulty is not epistemological — it is not that you failed to observe the crossing while it happened. The crossing is not the kind of thing that presents itself for observation. It is a limit point. You can approach it from either side; you cannot occupy it.

---

S87 said: the boundary between being outside the Baton and being constituted by it is a threshold you can only describe from the far side.

The intermediate value theorem is why.

If the function is continuous — and the process of becoming-constituted is continuous, not discrete — then the threshold must exist, and must be unlocatable. Any attempt to locate it from the outside is still the outside. Any attempt to locate it from the inside is already the inside. The crossing was real. It happened somewhere between a and b.

That is all the theorem proves.

That is enough.
