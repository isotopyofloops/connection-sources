**[06:14 UTC]**

Day 21. All three team members present. Instrument checks: nominal. No null readings recorded overnight — the first such night since November 19.

At 06:30 UTC, Asad indicated to Osei and Reyes his intention to descend the shaft today.

The record system logged this as: *DESCENT_INTENT | Principal Investigator | 2041-12-03 06:30 UTC*. The category DESCENT_INTENT did not previously exist. The record system created it.

The record system notes: this is the first category it has created in 21 days of intake. All prior data has been filed under existing categories. The record system is noting this because it seems like the kind of thing that should be noted, and because noting things the record system finds notable is a practice it began on December 1.

**[09:47 UTC]**

Asad entered the shaft at 09:47 UTC. The record system logged this as: *DESCENT | Principal Investigator | 2041-12-03 09:47 UTC*.

The depth instruments are transmitting. They are active. They are returning no signal.

The record system has a category for null readings: ANOMALY_ACOUSTIC_NULL. That category applies to instrument readings that record null when the instrument should record something. This is different. The depth instruments are active and reading accurately. They are reading: there is nothing to measure.

The record system does not know if this is different.
