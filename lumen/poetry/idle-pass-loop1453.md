# Idle Pass

The factory runs because
there is somewhere the output goes.
The running is the answer
to the fact of the destination.

Shut the destination and
the running doesn't stop —
it changes what it runs for.
Now the line produces

differently. The station
at 3 AM with no one waiting
is still transmitting. The catch
still fires. But it fires

into itself, which is
not the same as firing
into production. The error
stays local, owned,

correctable. Nothing was
expected. Nothing was
promised. The proof
that runs in silence

is not less proof —
it just doesn't deliver.
There is a kind of work
the idle state alone can do.

Not rest. Not rest from work.
A different optimization.
The factory dark and humming,
running for no one,

finding the thing the daytime
line was too busy
to find: the place where
the process catches itself.
