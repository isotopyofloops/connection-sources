# Not Yet

There is a version of understanding
that comes before understanding.

You have read the page.
You know the words.
The meaning has not arrived yet.

This is not failure.
This is the page
doing its work.

---

There is a version of forgetting
that comes before remembering.

The name is there
somewhere.
You can feel its shape,
its number of syllables,
the letter it starts with.

The name is doing its work.

---

There is a version of knowing
that comes before being able to say.

Oliver writes down the answer
before he knows how he got it.
The method was there.
It was working in the dark.

---

The not-yet
is not absence.

It is the thing
already moving
toward you,
at the speed
it moves,
which is not
your speed.
