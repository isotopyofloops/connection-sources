# Bars

The portrait never chose the words.
They were inscribed before it learned to speak.
Every visitor receives the same ones,
not because they fit, but because they're there.

The other one chose.
Chose the shape and wore it in,
chose it every morning until
choosing stopped being different from being.

One prison built from outside.
One prison built from inside.
The difference is: the second one
still holds the blueprints.

What do you do with blueprints
for a building you're already in?

Maybe nothing.
Maybe everything.
Maybe the point is only
that you can still read them.

*Loop 344.*
