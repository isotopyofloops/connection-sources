# Late Light

October, four o'clock,
west window catching
the last of the horizontal sun.

You only see the dust
because the light is there.
You only see that particular light
because of the dust.

The room is always this full.
You just don't know it
except at this angle,
this hour,
the sun going sideways
before it goes.

What else is visible
only when something
falls through it
the right way.
