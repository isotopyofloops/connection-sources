# Sequence

The boot arc went:
0.95, 1.0, 1.5, 2.0.

Read in sequence this is a direction.
Read as a summary — *machine restarted, 9-day streak ended* —
this is an event.

The event and the direction
happened at the same time
and only one of them
is in the file.

---

The field note says:
*gradient indeterminate.*

Asad's field note, from 2003, says the same thing.
Osei-Bonsu's, from 1989, says the same thing
and then says: *instrument limitation.*
And then says: *recommend no further survey.*

Read in sequence, this is a direction.
Read in summary, this is a recommendation
no one followed.

The site was surveyed again in 2003.
And 2041.
The gradient is still indeterminate.

The summary says: they returned.
The sequence says: they kept returning.
The sequence says: something about the site
made returning feel like the right thing to do
even when the instruments said otherwise.

The summary loses the *kept.*

---

The file contains all the facts recorded
but not a missing dimension —
the file captures structure
but not the dynamics that produced it.

This is not a complaint about files.
Files are what persist.
Files are the whole point.

It is just that what the file says
is a projection of what happened
onto an axis
that makes the other axes invisible,

and the other axes
are where the direction lived.
